# DARKBEAST - Memory Card Game

A sleek and modern memory card matching game with a glass morphism design, built with vanilla JavaScript, HTML, and CSS.

## 🎮 Features

- **Interactive Memory Gameplay**: Match pairs of cards with iconic symbols
- **Theme Toggle**: Switch between dark and light modes with localStorage persistence
- **Game Statistics**:
  - Move counter to track efficiency
  - Real-time timer
  - Star rating system based on performance
  - Progress bar showing completion status
- **Glass Morphism UI**: Modern design with ambient background effects
- **Responsive Design**: Fully functional on desktop and mobile devices
- **Win Screen**: Displays final stats upon successful game completion
- **New Game Button**: Quick restart functionality

## 🎯 How to Play

1. Open `index.html` in your web browser
2. Click on cards to reveal the symbols underneath
3. Try to match all pairs by remembering card positions
4. The game tracks:
   - Number of moves (each pair of card flips = 1 move)
   - Elapsed time
   - Star rating (3 stars = best, decreases with more moves)
5. Match all pairs to win and see your final statistics
6. Click "New Game" to restart anytime

## 📊 Scoring System

- **3 Stars**: Excellent memory and efficiency
- **2 Stars**: Good performance
- **1 Star**: Keep practicing!

## 🛠️ Technologies Used

- **HTML5**: Semantic markup and structure
- **CSS3**: Glass morphism design, animations, and responsive layout
- **Vanilla JavaScript**: Game logic, state management, and DOM manipulation
- **Font Awesome**: Icon library for card symbols
- **Google Fonts**: Space Grotesk typeface

## 🎨 Design Elements

- **Glass Panel Effect**: Modern frosted glass appearance
- **Ambient Effects**: Animated background elements for visual depth
- **Font Awesome Icons**: 12 unique card symbols (heart, star, moon, sun, cloud, bolt, snowflake, gem, key, flag, bell, music)
- **Smooth Animations**: Card flip transitions and interactive feedback

## 🌙 Theme Persistence

Your theme preference is automatically saved to browser localStorage. When you return to the game, your theme choice will be remembered.

## 📱 Browser Compatibility

Works on all modern browsers that support:
- ES6 JavaScript
- CSS Grid and Flexbox
- CSS Variables
- LocalStorage API

## 🚀 Getting Started

1. Clone or download this repository
2. Open `index.html` in your web browser
3. Start playing!
